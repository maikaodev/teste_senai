## Clonando o repositório

Via HTTPS

```bash
## Clone o projeto
git clone https://github.com/maikaodev/teste_senai.git

## Entre na pasta
cd teste_senai

## Instale as dependências
npm i ou npm install

## Execute a aplicação
npm run dev

## Servidor rodando - http://localhost:5173/
```
